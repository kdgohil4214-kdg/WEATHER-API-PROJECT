# 🌦️ Weather App (Atmosphere Check)

A simple and responsive **Weather Web Application** that allows users to check real-time weather details by entering a city name.

---

live project link = https://moonlit-entremet-774901.netlify.app/

## 🚀 Features

* 🌍 Search weather by city name
* 🌡️ Displays current temperature (°C)
* ☁️ Dynamic weather icons (cloud, rain, clear)
* 💧 Shows humidity level
* 🌬️ Displays wind speed
* ❌ Handles invalid city input
* 📱 Responsive and clean UI

---

## ⚙️ How It Works

1. User enters a city name
2. App sends request to OpenWeatherMap API
3. Weather data is fetched and displayed:

   * Temperature
   * City Name
   * Weather Icon
   * Humidity
   * Wind Speed

---

## 🔑 API Used

* OpenWeatherMap API
* Endpoint:

```
https://api.openweathermap.org/data/2.5/weather
```

---

## ▶️ How to Run

1. Clone this repository:

```
git clone https://github.com/your-username/weather-app.git
```

2. Open `index.html` in your browser

3. Enter a city name and click **"check it"**

---

## ⚠️ Important Note

* Replace the API key in `script.js` with your own:

```
appid=YOUR_API_KEY
```

You can get a free API key from:
👉 https://openweathermap.org/api
